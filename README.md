# hand_sign_recognition

Instructions on how to run:

1.  Download hand sign dataset from https://www.kaggle.com/grassknoted/asl-alphabet
2.  Unzip the folder and the contents within it
3.  Put folder in the same directory as cnn.py
4.  Rename asl_alphabet_train to training_set
5.  In the same directory as training_set, create an empty folder called test_set
6.  Run "python cnn.py"
